About MyToDo

Welcome to MyToDo, your simple and efficient to-do list manager built with Flask and Bootstrap.

This app helps you keep track of your daily tasks and stay organized. You can add new tasks, update existing ones, and delete tasks you've completed.

MyToDo was created to demonstrate how to build a full CRUD web application using Flask and SQLite.

Features

  • Add new to-do items with titles and descriptions.
  • Update existing to-do items easily.
  • Delete tasks you no longer need.
  • Responsive and clean UI with Bootstrap 5.
